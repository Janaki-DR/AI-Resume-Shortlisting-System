# AI-Resume-Shortlisting-System
This project is an AI-based hiring assistant designed to help HR teams quickly screen candidate resume and identify suitable applicants based on required skills.
## Problem
Companies receive hundreds of resumes for a single job role. Manual screening takes time and there is a chance that qualified candidates may be missed.
## Solution
This system analyzes resume text and compares it with the required skills for the selected job role. It calculates a match percentage and suggests whether the candidate should be shortlisted.
## Working
1. HR selects a job role.
2. Candidate resume text is entered.
3. The system compares the resume skills with required skills.
4. A match percentage is calculated.
5. If the score meets the threshold, the candidate is shortlisted.
